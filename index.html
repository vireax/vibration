Project description

This project aims to solve the problem of vibration based damage detection in truss structure. By using co-operative co-evolutionary genetic algorithms, the author expect faster and more accurate optimization than conventional genetic algorithms. The current status is in programming stage. I'm rewriting the draft of each chapter. Previous notes were also imported into this site to explore possible path during the research.
